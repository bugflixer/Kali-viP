# Kali-viP
This tool is helps you to install kali linux in non-rooted device.. 😂


## INSTALLATION [Termux] :

* `apt-get update -y`
* `apt-get upgrade -y`
* `pkg install python -y`
* `pkg install python2 -y`
* `pkg install git -y`
* `pip install lolcat`
* `git clone https://github.com/bugflixer/Kali-viP.git`
* `cd Kali-viP`
* `bash start`
* `type yes`
* `choose 2 to enter key otherwise choose 1 `
* `End 🙏 enjoY`
